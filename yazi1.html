Sıfırdan GitHub Pages ile Blog Kurma Rehberi

Yayınlanma: 9 Haziran 2026 • Okuma Süresi: 3 dk

GitHub Pages, statik web sitelerinizi tamamen ücretsiz bir şekilde tüm dünya ile paylaşmanızı sağlayan harika bir servistir. Üstelik SSL sertifikası (https) otomatik olarak tanımlanır.

Bu yöntemin en güzel yanı, karmaşık veritabanı ayarlarıyla, sunucu güncellemeleriyle veya güvenlik açıklarıyla uğraşmak zorunda kalmamanızdır. Siteniz doğrudan GitHub sunucularından beslendiği için adeta ışık hızında açılır.

Kendi yazılarınızı eklemek için tek yapmanız gereken bu şablona benzer HTML dosyaları oluşturmak ve ana sayfanızdan link vermektir.
