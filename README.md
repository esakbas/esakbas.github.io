# 🌿 Hindiba - Eşref Akbaş Blog

> Belki de şimdiye kadar ardından söylenebilecek en zarif benzetme "Hindiba" idi…

## 📖 Hakkında

Hindiba, Eşref Akbaş'ın edebiyat, şiir ve günlük yazılarını paylaştığı kişisel blogudur.

## 🚀 Özellikler

- 📱 **PWA Desteği** - Mobil cihazlarda uygulama gibi çalışır
- 🌓 **Koyu/Açık Tema** - Otomatik ve manuel tema değişimi
- 🔍 **Arama** - Yazılar arasında hızlı arama
- 🏷️ **Kategori Filtreleme** - Kategorilere göre filtreleme
- 💬 **Yorum Sistemi** - Firebase ile yönetilen yorumlar
- 📊 **İstatistikler** - Blog istatistikleri
- ♾️ **Sonsuz Kaydırma** - Otomatik yüklenen yazılar

## 🛠️ Teknolojiler

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **PWA**: Service Worker, Web App Manifest
- **Veritabanı**: Firebase Firestore
- **API**: GitHub Issues API
- **Icons**: Font Awesome 6
